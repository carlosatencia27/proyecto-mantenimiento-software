# \# LogiFleet Colombia

# 

# Sistema de gestión de flotas de vehículos para empresas de transporte.

# 

# \## Descripción

# 

# LogiFleet es una plataforma que permite a empresas de transporte gestionar sus flotas de vehículos, hacer seguimiento en tiempo real, generar reportes y programar mantenimientos.

# 

# \## Funcionalidades

# 

# \- Registro de vehículos

# \- Seguimiento GPS en tiempo real

# \- Generación de reportes

# \- Alertas de mantenimiento

# \- Asignación de conductores

# 

# \## Tecnologías

# 

# \- HTML5

# \- CSS3

# \- JavaScript

# 

# \## Cómo usar

# 

# 1\. Abre el archivo `index.html` en tu navegador

# 2\. El sistema cargará automáticamente

# 3\. Usa los botones del menú para navegar
